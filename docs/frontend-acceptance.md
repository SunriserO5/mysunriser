# 前端最小验收说明

## 1. 启动方式

### 启动后端

在项目根目录执行：

```bash
cd backend
./mvnw spring-boot:run
```

默认监听 `http://localhost:8080`。

### 启动前端开发服务器

在项目根目录执行：

```bash
cd frontend
npm run dev
```

默认访问 `http://localhost:5173`。

## 2. 联调验证路径

1. 首页：`/`
2. 博客列表：`/blog`
3. 文章详情：`/blog/:slug`
4. 关于页：`/about`

## 3. 关键交互验收点

1. 列表页分页可切换，URL 会保留 `?page=` 参数。
2. 切换分页后自动滚动到页面顶部。
3. 列表页加载失败时展示“重试”按钮。
4. 详情页加载失败时展示“重试”按钮。
5. 文章详情页标题会动态更新为 `文章标题 | MySunriser`。
6. 首页主按钮和右上角导航在正常态与悬停态均应保持可读。

## 4. 已知限制

1. `npm run preview` 默认不代理 `/api`，仅用于预览构建产物。
2. 异常“超时”场景依赖网络和后端行为，本地可通过断开后端服务模拟。
